# Qualis CAPES – Dados de Periódicos

Este repositório reúne avaliações de quadriênios do **Qualis Periódicos (CAPES)** obtidas a partir da plataforma Sucupira:

https://sucupira-legado.capes.gov.br/sucupira/public/consultas/coleta/veiculoPublicacaoQualis/listaConsultaGeralPeriodicos.jsf

O objetivo é facilitar o acesso e a análise desses dados.

---

## Sobre o Qualis

O Qualis classifica periódicos usados por programas de pós-graduação no Brasil em estratos como:

A1, A2, A3, A4, B1, B2, B3, B4 e C.

---

## Atenção ao interpretar os dados

- As classificações são **definidas por área do conhecimento**
- Diferentes áreas têm **padrões distintos de publicação e citação**
- O **fator de impacto varia muito entre áreas**

Isso significa que:

- Um A1 em uma área não é equivalente a um A1 em outra
- Comparações diretas entre áreas podem ser enganosas

---

## Aviso importante

A CAPES anunciou a **descontinuidade do Qualis Periódicos**:

https://www.gov.br/capes/pt-br/assuntos/noticias/sobre-o-qualis-periodicos-na-avaliacao-quadrienal-2021-2024

Assim, estes dados devem ser usados como:

- Referência histórica
- Base para análises retroativas
